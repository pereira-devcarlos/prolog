# Estudos em Programação Lógica com Prolog

Este repositório contém materiais, exemplos e exercícios relacionados ao estudo de Programação Lógica utilizando a linguagem Prolog. O objetivo é práticar e consultar sobre conceitos fundamentais e aplicações de Prolog.

## Sobre Prolog

Prolog (Programming in Logic) é uma linguagem de programação declarativa baseada em lógica, muito utilizada em inteligência artificial, processamento de linguagem natural e resolução de problemas que envolvem conhecimento e inferência lógica.

## Estrutura do Repositório

- Exemplos de código em Prolog
- Exercícios resolvidos e propostos
- Anotações teóricas sobre lógica de programação
- Referências e materiais de apoio

## Como utilizar

1. Instale um interpretador Prolog, como SWI-Prolog ([https://www.swi-prolog.org/](https://www.swi-prolog.org/)).
2. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```
3. Navegue até os arquivos de exemplo e exercícios e execute-os no interpretador Prolog de sua preferência.
